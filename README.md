# mini_pot  
U4-ER_project (smart green house)  
  
## 팀원  
phj, kjs  
  
## 구성  
* temp 폴더 : 테스트 or 추후 사용가능 코드  
    + .idea : 파이참 설정 폴더 (사용 X)  
    + log_* : 예전 기록 + 추후 참고가능  
    + python_test.py : 파이썬 구동 테스트 (단순 print)  
* blueprint 폴더 : 설계도면 모음  
* log 폴더 : 기록 모음  
* module_code 폴더 : 일부분 완성되어 추후 통합할 코드 모음  
  
* > sql_server.py : 현제 작업중
